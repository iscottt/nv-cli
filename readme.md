# 🚀 主题开发脚手架工具

一个功能强大、界面美观的脚手架工具，帮助开发者快速创建各种类型的项目模板。支持编辑器块、通用插件和多种主题开发模式。

![CLI Demo](https://img.shields.io/badge/CLI-专业版-blue?style=for-the-badge)
![Node 版本](https://img.shields.io/badge/Node.js-12+-green?style=for-the-badge)
![许可证](https://img.shields.io/badge/许可证-MIT-yellow?style=for-the-badge)

## ✨ 特性

- 🎨 **美观的终端界面** - 使用渐变色彩、图标和优雅的边框设计
- 📦 **多种模板选择** - 支持5种不同类型的项目模板
- ⚡ **快速创建** - 一键生成完整的项目结构
- 🔧 **高度可配置** - 每个模板都有优化的默认配置
- 🛠 **现代化工具链** - 集成最新的开发工具和构建系统
- 📱 **响应式设计** - 主题模板支持移动端优先

## 📋 支持的模板

| 模板类型 | 描述 | 技术栈 |
|---------|------|--------|
| 🎨 **编辑器块模板** | 用于创建富文本编辑器组件 | 原生JS/React组件 |
| 🔌 **通用插件模板** | 用于创建可复用的插件 | 通用JS/TypeScript |
| 🖥️ **主题开发-客户端渲染** | 基于React/Vue的纯前端主题 | Vite + React/Vue + TailwindCSS |
| 🚀 **主题开发-服务端渲染** | 基于Next.js的服务端渲染主题 | Next.js + React + TailwindCSS |
| 🌐 **主题开发-前后端混合渲染** | 基于Nuxt 3的通用渲染主题 | Nuxt 3 + Vue 3 + TailwindCSS |

## 🚀 快速开始

### 安装

```bash
# 通过 npm 全局安装
npm install -g @iscott/nv-cli

# 或使用 npx 直接运行
npx @iscott/nv-cli

# 启动脚手架
nv-cli
```
